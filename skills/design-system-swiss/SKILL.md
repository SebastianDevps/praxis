## When to Use

Editorial sites, documentation, technical content, portfolios where the writing carries the weight. When typography IS the design.

## Key Characteristics

- Strict column grid (4–12 cols depending on breakpoint); no free-floating elements.
- Type sets the visual hierarchy — size and weight only, minimal color.
- Generous whitespace: breathing room between sections is a design element, not waste.
- 1–2 accent colors maximum; black/dark gray does most of the work.
- No decorative illustration or gradient heroes — content and grid carry the page.

## Key Tokens

```
Type scale: 5–7 steps, large jumps between display and body
Grid: 12-col with defined gutters and margins
Color: near-black primary, 1 accent, off-white bg
Spacing: generous — vertical rhythm from type line-height
Radius: 0 or very small (≤ 4px) — sharp edges reinforce structure
```

## Tradeoff

Cold/technical if not balanced with warmth in copy or photography. Works best when the content is genuinely interesting.
