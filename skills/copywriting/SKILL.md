## Hierarchy of Clarity

1. **Functional** — user knows exactly what will happen
2. **Concise** — fewest words that preserve meaning
3. **On-brand** — voice, not decoration

If 1 and 3 conflict, 1 wins.

## UI Microcopy

- CTAs: verb first — "Save changes", not "Changes saved confirmation"
- Verb options ranked: Do → Start → Get → View → Open — avoid "Submit" (bureaucratic)
- Empty states: explain WHY it's empty + one action to fill it
- Error messages: what went wrong + what to do, no blame ("Something went wrong" is not an error message)
- Tooltips: add only when the label leaves real ambiguity; never restate the label

## Marketing Copy

- Concrete over vague: "Deploys in 30 seconds" beats "Fast deployment"
- Headline = outcome the user wants, not what the product does
- One message per surface — resist the urge to add a second claim
- Social proof: specifics ("4 000 teams") beat superlatives ("thousands")

## Anti-Cliché Checklist

Strip these before delivering — they signal slop:

| Banned | Replace with |
|--------|-------------|
| Elevate | Improve / raise / lift |
| Seamless | Smooth / one-step / direct |
| Unleash | Use / unlock / enable |
| Delve | Look into / explore / check |
| Transform | Change / rethink / rebuild |
| Empower | Let / help / give control to |
| Leverage | Use / apply |
| Game-changer | — (show, don't claim) |

## Brand Voice Matching

Before writing, note three things: tone (formal/casual), person (1st/2nd), density (terse/warm). Hold those throughout. Switching mid-surface feels schizophrenic.
